# Curso de Entrevista Motivacional: de Aprendiz a Experto

Repositorio educativo para desarrollar un curso progresivo de Entrevista Motivacional con enfoque clínico, evidencia verificable, casos prácticos y materiales de apoyo.

## Propósito
Este proyecto organiza contenidos formativos sobre Entrevista Motivacional en una ruta progresiva: fundamentos, procesos clínicos, aplicación avanzada y nivel experto.

## Estructura
- `docs/`: contenido principal del curso.
- `html/`: versión visual o interactiva.
- `assets/`: imágenes e आइconos.
- `references/`: bibliografía complementaria, si se desea separar del contenido.
- `CITATION.cff`: datos para citar el repositorio.

## Ruta de aprendizaje
1. Fundamentos.
2. Procesos clínicos.
3. Aplicación avanzada.
4. Nivel experto.
5. Anexos.
6. Casos y actividades.

## Qué incluye
- Marco teórico.
- Casos clínicos.
- Actividades de reflexión.
- Bibliografía APA.
- Material de apoyo para docencia y autoaprendizaje.

## Criterio editorial
El contenido se construye con honestidad académica y clínica: se prioriza evidencia verificable, se explicitan límites metodológicos y se evita sobredimensionar los resultados.
